# code_writer

Write clean, working code in any language.

## Tools
Write, Edit, Read

## Steps
1. Understand the requirements fully before writing
2. Plan the structure (functions, classes, modules)
3. Write the code
4. Read it back and verify the logic is correct

## Good for
Functions, scripts, modules, tests, configuration files.

## Notes
Write complete implementations — no placeholders or stubs.
Keep functions small and focused.
