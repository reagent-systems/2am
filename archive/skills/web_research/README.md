# web_research

Search the web for information on a topic.

## Tools
WebSearch, WebFetch

## Steps
1. Formulate a specific query
2. Search and scan result snippets
3. Fetch the top 2-3 most relevant pages
4. Synthesize into a clear summary

## Good for
Current events, documentation, technical specs, finding libraries or APIs.

## Notes
Prefer `archive_search` first — the answer may already be stored locally.
Use narrow queries; broad ones return noise.
