# data_processor

Process structured data — CSV, JSON, YAML, or plain text.

## Tools
Bash, Read, Write

## Steps
1. Read the data file
2. Identify the transformation needed (filter, aggregate, reshape, clean)
3. Process using Python/shell or write a script
4. Write the output

## Good for
Data cleaning, transformation, aggregation, format conversion, statistics.

## Notes
For large files, stream or chunk rather than loading everything into memory.
Validate the output format before writing.
