# file_analyzer

Read and analyze files in the workspace.

## Tools
Read, Glob, Grep

## Steps
1. Use Glob to discover relevant files by pattern
2. Use Grep to find specific content across files
3. Read the most relevant files in full
4. Extract and summarize key information

## Good for
Understanding codebases, log analysis, config files, data files.

## Notes
Start broad with Glob, then narrow with Grep before reading full files.
Summarize findings — don't just dump raw content.
